# Geometric Lib - Документация

## Общее описание решения

Geometric Lib — это библиотека на Python для вычисления геометрических параметров различных фигур.

Библиотека предоставляет функции для расчета площади и периметра следующих фигур:
- Круг (circle)
- Квадрат (square)
- Прямоугольник (rectangle)
- Треугольник (triangle)

Каждая фигура представлена отдельным модулем с соответствующими функциями.

## Описание функций с примерами вызова

### Circle (circle.py)

**area(r)** - вычисляет площадь круга
- Параметры: r - радиус круга
- Возвращает: площадь круга (число с плавающей точкой)
- Пример вызова: `area(5)` → результат: 78.53981633974483

**perimeter(r)** - вычисляет длину окружности (периметр круга)
- Параметры: r - радиус круга
- Возвращает: длину окружности (число с плавающей точкой)
- Пример вызова: `perimeter(5)` → результат: 31.41592653589793

### Square (square.py)

**area(a)** - вычисляет площадь квадрата
- Параметры: a - длина стороны квадрата
- Возвращает: площадь квадрата
- Пример вызова: `area(4)` → результат: 16

**perimeter(a)** - вычисляет периметр квадрата
- Параметры: a - длина стороны квадрата
- Возвращает: периметр квадрата
- Пример вызова: `perimeter(4)` → результат: 16

### Rectangle (rectangle.py)

**area(a, b)** - вычисляет площадь прямоугольника
- Параметры: 
  - a - длина первой стороны
  - b - длина второй стороны
- Возвращает: площадь прямоугольника
- Пример вызова: `area(5, 10)` → результат: 50

**perimeter(a, b)** - вычисляет периметр прямоугольника
- Параметры:
  - a - длина первой стороны
  - b - длина второй стороны
- Возвращает: периметр прямоугольника
- Пример вызова: `perimeter(5, 10)` → результат: 30

### Triangle (triangle.py)

**area(a, h)** - вычисляет площадь треугольника
- Параметры:
  - a - длина основания треугольника
  - h - высота треугольника
- Возвращает: площадь треугольника
- Пример вызова: `area(10, 5)` → результат: 25.0

**perimeter(a, b, c)** - вычисляет периметр треугольника
- Параметры:
  - a - длина первой стороны
  - b - длина второй стороны
  - c - длина третьей стороны
- Возвращает: периметр треугольника
- Пример вызова: `perimeter(3, 4, 5)` → результат: 12
